Features

These are some of the unique and awesome features that Apollo has.

Custom Database

Apollo's custom database is a custom made database for Apollo, specfically strike and message count commands.
It is primarily built using MongoDB and the mongoose NPM module.

Bug Reporting

Apollo has the unique way of making sure that every bug or glitch in Apollo is fixed.
This is done by being able to easily being able to contact the developer with any bugs or glitches by using the -reportbug [bug] command.

Uptime

Apollo has fantastic uptime, with multiple hosting techniques, so that if the main VPS (Virtual Private Server) goes down, Apollo can be hosted on one of the other VPSes, so that Apollo can never be offline, and our users can never lose out.
