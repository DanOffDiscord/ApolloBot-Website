Commands

This is a list of commands that Apollo has.



Command Name Description
about Posts an embed with basic information about the bot.
eval Allows you to evaluate Javascript code. Developers Only
help DMs you this help menu.
ping Pings the connection of the bot, and returns a millisecond value.
serverinfo Posts an embed with basic information about the server.
invite Posts an embed with the invite link so you can add Apollo to your server. (this link)
reportbug Posts an embed with details of how to report a bug in Apollo.
poll Allows you to make a poll, with thumbs up and thumbs down reactions
8ball Allows you to ask the magic 8ball a question.
shutdown Shuts the bot down at it's host location. Developers Only
reload Allows you to reload the bot. Developers Only
